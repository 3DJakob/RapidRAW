# RapidRAW Iced POC

Small native UI proof of concept for exploring an `Iced`-based RapidRAW shell.

## Included

- Project home screen
- Editor screen
- Large image preview
- Horizontal filmstrip sourced from images in `../public`

## Run

```bash
cargo run --manifest-path iced-poc/Cargo.toml
```

## Click To Run

On macOS, you can also double-click:

`Build and Run Release.command`

That opens Terminal, builds the release binary, and launches it once the build finishes.

If the sample project appears empty, add a few `.jpg`, `.jpeg`, `.png`, or `.webp` files to the repo's `public/` directory and relaunch.
